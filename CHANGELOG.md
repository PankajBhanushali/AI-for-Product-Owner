## [1.0.0] - 2026-02-24

### Added
- Complete 20-week AI certification curriculum
- 7 learning phases:
  - Phase 1: AI Foundations (3 weeks)
  - Phase 2: AI Strategy (3 weeks)
  - Phase 3: Product Execution (4 weeks)
  - Phase 4: Data & Model (3 weeks)
  - Phase 5: Governance & Risk (3 weeks)
  - Phase 6: Scaling (2 weeks)
  - Phase 7: Capstone (2 weeks)
- Comprehensive week-by-week content with learning objectives and exercises
- Professional website with responsive design
- Phase hub pages with week navigation
- Breadcrumb navigation for easy traversal
- Labs and templates:
  - AI Backlog Template
  - AI Cost Calculator Guide
  - AI Risk Register Template
  - Prompt Test Matrix
- Certification exam framework
- Code of Conduct for community
- Open source contribution guidelines

### Changed
- Website deployed to GitHub Pages
- Content organized hierarchically by phase
- Navigation structure optimized for discovery

### Features
- ✅ Desktop and mobile responsive design
